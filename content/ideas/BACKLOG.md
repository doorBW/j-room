# 콘텐츠 백로그 (발행 주제 풀)

> 발행할 주제들의 마스터 목록. 떠오를 때마다 추가하고, 만들면 체크.
> 카테고리 정의는 [docs/CONTENT_TYPES.md](../../docs/CONTENT_TYPES.md),
> 발행 캘린더는 [docs/CONTENT_STRATEGY.md](../../docs/CONTENT_STRATEGY.md) 참고.

---

## 📦 현재 재고 상태

```
🟢 재고(ready): 2개 / 목표 4개 — 2개 더 만들면 발행 시작
   - ✅ PLAY33 건대 3테마 추천 (ready · 팩트체크 완료 2026-06-15)
   - ✅ 룸즈에이 메가게임 후기 (ready)
```
> 발행 대기(status: ready) 콘텐츠 개수. 4개 이상 유지가 목표.

---

## 🎯 발행 시작 플랜 (확정)

**전략: 추천 위주로 재고 4개 확보 후 주 2회 시작** (격주 캘린더)
- 카테고리 균등 X → **총 재고 4개**가 목표 (만들기 쉬운 추천 위주로 빠르게)
- 연방·소식은 발행 시작 후 병행 — 빈 슬롯은 추천으로 대체 (유연성 규칙)
- 소식은 AI 집계 워크플로우부터 구축 필요 (지금은 미구축)

**제작 큐** (재고 4개 채우기):
1. ✅ 추천 — PLAY33 건대 3테마 *(완성)*
2. ✅ 후기 — 룸즈에이 메가게임 *(완성)*
3. ⬜ 추천 또는 연방 — 미정 ← **다음 제작**
4. ⬜ 미정

→ 4개 모이면 발행 시작. 시작 후엔 "발행 1개당 1개 보충" 페이스로 재고 유지.

---

## ✅ 사용법

- 주제가 떠오르면 해당 카테고리에 `- [ ] 주제` 추가
- 만들기 시작하면 → `content/drafts/` 로 옮기고 여기선 `[~]` 표시
- 발행 완료되면 → `[x]` 체크 + 날짜
- ⭐ = 우선순위 (먼저 만들 것)

---

## 1. 장르/주제별 추천 (recommendation) — 메인 ⭐

> 장르 또는 주제별 방탈출 2~5개 큐레이션. 400개 경험 풀에서 계속 뽑을 수 있는 주력 콘텐츠.

### 장르별
- [ ] ⭐ 진짜 무서운 공포 방탈출 (TOP n)
- [ ] ⭐ 머리 굴리는 추리 방탈출
- [ ] 문제풀이/장치 중심 방탈출
- [ ] 판타지 세계관 방탈출
- [ ] 코믹/유쾌한 방탈출
- [ ] 감성/스토리 끝판왕 방탈출
- [ ] 스릴러/액션 방탈출

### 주제별
- [ ] ⭐ 방탈출 입문자에게 추천하는 첫 테마
- [ ] 공포 없이 즐기는 안전 방탈출
- [ ] 2인(커플/단짝) 추천
- [ ] 4인이 함께 풀기 좋은 방탈출
- [ ] 90분 이하 가볍게 / 대작(2시간+)
- [ ] 가성비 좋은 방탈출
- [ ] 재방문하고 싶은 인생 테마

### 지역별
- [ ] 강남 BEST
- [ ] 홍대 BEST
- [ ] 건대 BEST
- [ ] 신촌/이대 BEST
- [ ] (지방) 부산/대구/광주 명작

> 💡 추천 개수는 2~5개 유연하게. 너무 많으면 카드 길어짐. 3~5개 권장.

## 2. 연방 코스 (course) — 메인

> 위치 기반 연속 방탈출 타임라인. 본인이 순서·시간 제공. 제작 부담 크니 과거 코스부터.

- [ ] ⭐ 강남역 4시간 3연방 코스
- [ ] 홍대 방탈출 + 맛집 코스
- [ ] 건대 하루 코스
- [ ] 신촌/이대 입문자 코스
- [ ] 강남 공포 특화 코스
- [ ] 데이트 코스 (방탈출 + 카페 + 식사)
- [ ] 비 오는 날 실내 코스
- [ ] 4인 회식 대체 코스
- [ ] (지방) 부산 서면 코스

> 💡 동선 5분 이내 + 난이도 그라데이션(가벼움→무거움)이 베스트.

---

## 3. 방탈출 소식 (news) — 정기

> 주간 신규 오픈/테마/이벤트/예약 소식 집계. AI 활용. 사실 정확성 필수.
> 워크플로우는 [CONTENT_STRATEGY.md §5](../../docs/CONTENT_STRATEGY.md) 참고.

- [ ] ⭐ 이번 주 새로 오픈한 테마 모음 (정기 코너)
- [ ] 이번 달 신규 테마 총정리 (월간)
- [ ] 예약 전쟁 — 오픈런 해야 하는 인기 테마
- [ ] 이벤트/할인 진행 중인 매장
- [ ] (관심 매장) 신규 테마 예고 소식

> 💡 정기 코너라 매주 같은 포맷 재사용. 템플릿 한 번 만들면 채우기만.

---

## 4. 방탈출 후기 (review) — 비정기

> 직접 다녀온 테마. 당분간 과거 회상으로, 이후 다녀올 때마다 추가.

- [ ] ⭐ (인생테마 1 — 매장/테마명)
- [ ] ⭐ (인생테마 2)
- [ ] ⭐ (최근 다녀온 테마)
- [ ] (강한 공포로 기억에 남는 테마)
- [ ] (추리/장치가 훌륭했던 테마)
- [ ] (감성/스토리가 좋았던 테마)
- [ ] (숨은 명작 / 저평가 테마)
- [ ] (호불호 갈리는 테마 — 솔직 후기)

> 💡 후기는 "최근 다녀온 순"으로 쌓아도 됨. 별점 기준은 BRAND_GUIDE 참고해 일관되게.
