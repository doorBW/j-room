# 콘텐츠 백로그 (발행 주제 풀)

> 발행할 주제들의 마스터 목록. 떠오를 때마다 추가하고, 만들면 체크.
> 카테고리 정의는 [docs/CONTENT_STRATEGY.md](../../docs/CONTENT_STRATEGY.md) 참고.

---

## 📦 현재 재고 상태

```
🔴 재고(ready): 0개 — 첫 콘텐츠 생산 필요
```
> 발행 대기(status: ready) 콘텐츠 개수. 4개 이상 유지가 목표.

---

## ✅ 사용법

- 주제가 떠오르면 해당 카테고리에 `- [ ] 주제` 추가
- 만들기 시작하면 → `content/drafts/` 로 옮기고 여기선 `[~]` 표시
- 발행 완료되면 → `[x]` 체크 + 날짜 기록
- ⭐ = 우선순위 높음 (먼저 만들 것)

---

## A. 방탈출 후기 (review) — 주력 ⭐

> 직접 다녀온 테마는 전부 후보. 인생테마/최근 다녀온 것부터.

- [ ] ⭐ (인생테마 1 — 매장/테마명 적기)
- [ ] ⭐ (인생테마 2)
- [ ] ⭐ (최근 다녀온 테마)
- [ ] (강한 공포로 기억에 남는 테마)
- [ ] (추리/장치가 훌륭했던 테마)
- [ ] (감성/스토리가 좋았던 테마)
- [ ] (호불호 갈리는 테마 — 솔직 후기)
- [ ] (입문자에게 추천하는 첫 테마)
- [ ] (아쉬웠던 테마 — 완곡하게)
- [ ] (숨은 명작 / 저평가 테마)

> 💡 후기는 "최근 다녀온 순"으로 그냥 쌓아도 됨. 별점 기준은 BRAND_GUIDE 참고해서 일관되게.

## C. 테마별 추천 (recommendation)

- [ ] ⭐ 방탈출 입문자에게 추천하는 첫 테마 5선
- [ ] 진짜 무서운 공포 방탈출 5선
- [ ] 머리 굴리는 추리/논리 방탈출 5선
- [ ] 4인이 함께 풀기 좋은 방탈출 5선
- [ ] 2인(커플/단짝) 추천 방탈출 5선
- [ ] 감성/스토리 끝판왕 방탈출 5선
- [ ] 공포 없이 즐기는 안전 방탈출 5선
- [ ] 90분 이하 가볍게 즐기는 5선
- [ ] 대작(2시간+) 방탈출 5선
- [ ] 활동성 높은 모험형 방탈출 5선
- [ ] 가성비 좋은 방탈출 5선
- [ ] 재방문하고 싶은 인생 테마 5선

## D. 입문 가이드 / 꿀팁 (guide)

- [ ] ⭐ 방탈출 처음이세요? 알아두면 좋은 7가지

- [ ] 난이도 표기, 어떻게 봐야 할까
