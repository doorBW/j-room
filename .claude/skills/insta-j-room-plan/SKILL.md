## 1. 계정

- **핸들**: @jroom.official (프로페셔널/크리에이터)

- **화자**: 방탈출 400+ "늙크크" (스마일·메타몽·루피·동자모) — 공테(공포테마) 도파민에 빠진 팀

- **톤**: 친근한 솔직 위트, 스포 금지, 공식 표기 의심하되 출처 존중

## 2. 콘텐츠 카테고리 4종

| 카테고리 | 성격 | 생산 난이도 | 제작 스킬 |
|---------|------|-----------|----------|
| **추천** (장르/주제/지역별) ⭐ | 메인 주력 | 낮음 | `insta-j-room-create-recom` ✅ |

## 3. 발행 캘린더 (2주 격주 사이클)

```
        수 20시     토 11시     일 20시(보너스)
1주차 │  추천   │   연방   │   후기 (생산 시)
2주차 │  소식   │   후기   │   추천 (생산 시)
```

- **기본 주 2회**: 수 20시 / 토 11시
- **보너스**: 일 20시 (재고 있을 때만)
- 유연성: 재고 없는 슬롯은 **추천으로 대체**하거나 건너뛰기 (일관성 > 빈도)

> 상세: [docs/CONTENT_STRATEGY.md](../../../docs/CONTENT_STRATEGY.md)

## 4. 운영 원칙

1. **생산은 몰아서(배치), 발행은 규칙적으로** — 만드는 날 ≠ 올리는 날
2. **재고 2주치(4개) 유지** — 바쁜 주에도 안 끊기게
3. **빈 슬롯은 추천으로 메우거나 건너뛰기** — 억지로 올리지 말 것
4. 콘텐츠 상태: idea → draft → ready(재고) → scheduled → published

## 6. 사용 가능한 스킬

| 스킬 | 용도 | 상태 |
|------|------|------|
| `insta-j-room-create-recom` | 추천 카드뉴스 제작 | ✅ 사용 가능 |
| `insta-j-room-create-review` | 후기 카드뉴스 | ✅ 사용 가능 |
| `insta-j-room-create-course` | 연방 코스 카드뉴스 | ✅ 사용 가능 |
| `insta-j-room-create-news` | 소식 카드뉴스 (AI 집계) | ✅ 사용 가능 |
| `insta-j-room-plan` | 이 안내 (운영 길잡이) | ✅ 이 문서 |

> **카드2는 카테고리별로 다름**: 추천 = 늙크크 자기소개(고정) / 후기 = 별점 / 연방 = 타임라인 / 소식 = 이번 주 요약.
> 늙크크 브랜딩은 표지 kicker("늙크크가 모아온/다녀온/체험한")로 전 카테고리 공통 유지.

## 7. 생성 스킬 호출 가이드 ⭐

> **공통 원칙**: 본인만 아는 것(**체감 평가·한줄평·순서·코스**)만 주면 됨.
> 공식 정보(**난이도·가격·위치·포스터**)는 스킬이 웹에서 확보하고, 부족하면 되묻는다.
> 정보가 덜 갖춰져도 OK — 기억나는 대로 편하게 던지면 대화하며 다듬는다.

### 🎯 추천 — `/insta-j-room-create-recom`
한 매장 N테마 / 장르 TOP / 지역 BEST 등 2~5개 큐레이션 (8장).

| | 내용 |
|---|---|
| **꼭 주기** | ① 형식(한 매장 N테마 / 장르 / 지역) ② 매장·테마 2~5개 ③ 각 **체감 난이도·공포도** ④ 한줄평 ⑤ 추천 순서 |
| **자동(웹)** | 공식 장르·난이도·공포도, 인원, 시간, 가격, 위치, 포스터 |
| **팁** | 가장 빠름. "한 매장 N테마"가 제일 쉬움 (PLAY33·메가게임 방식) |

```
/insta-j-room-create-recom
홍대 OO매장 3테마 추천
- 테마A: 체감난이도 4, 공포 3.5/5, "연출 미쳤음" / 1순위
- 테마B: 체감난이도 3, 공포 2/5, "입문용 딱" / 3순위
- 테마C: 체감난이도 5, 공포 4/5, "두뇌 풀가동" / 2순위
```

### ⭐ 후기 — `/insta-j-room-create-review`
직접 체험한 **1개 테마** 깊이 리뷰 (7장).

| | 내용 |
|---|---|
| **꼭 주기** | ① 매장·테마 1개 ② 별점(난이도/공포도/추리비중/추천도) ③ 권장 인원 ④ 총평 ⑤ 좋았던 점 2~3 ⑥ 아쉬운 점 1~2 ⑦ 추천/비추천 대상 |
| **자동(웹)** | 가격, 위치, 소요시간, 포스터 |
| **팁** | 별점 기준 일관성 유지 · 스포 금지 · 별점 항목은 테마 특성에 맞게 조정 가능(예: 메가게임=팀워크) |

```
/insta-j-room-create-review
제로월드 강남 '비밀의화원' 후기
별점: 난이도4 공포2 추리5 추천5, 권장 2~3인
총평: "예상 못한 페이크 트릭"
좋았던: 장치 연결 매끄러움 / 후반 연출 한방
아쉬운: 초반 난이도 점프 큼
추천: 추리 좋아하면 / 비추: 공포 완전 못 하면
```

### 🗺️ 연방 코스 — `/insta-j-room-create-course`
한 지역 연속 방탈출 타임라인 + 동선 (7장).

| | 내용 |
|---|---|
| **꼭 주기** | ① 지역 ② 시간표(몇 시에 뭐) ③ 매장·테마 순서대로(+난이도/인원/시간) ④ 동선(도보 N분) ⑤ 총 예산 ⑥ 식사/카페(선택) |
| **자동(웹)** | 가격·주소 확인, 포스터 |
| **팁** | 동선 5분 이내 + 난이도 그라데이션(가벼움→무거움)이 베스트 |

```
/insta-j-room-create-course
강남역 4시간 3연방
10:30 A매장 테마1(난이도3, 2~3인, 60분) → 도보4분, 점심
13:30 B매장 테마2(난이도4, 3~4인, 75분) → 도보6분
17:00 C매장 테마3(난이도5, 4인, 80분)
1인 약 9만원
```

### 🗞️ 소식 — `/insta-j-room-create-news`
주간 신규 오픈·이벤트·예약 소식 정기 코너 (6~8장).

| | 내용 |
|---|---|
| **꼭 주기** | 소식 3~5건(매장/테마·구분·오픈일·위치·한줄) — 또는 **"이번 주 소식 모아줘"** 만 해도 됨 |
| **자동(웹)** | 집계 보조 + 출처 확인 (단 최종 팩트체크는 함께) |
| **팁** | ⚠️ 발행 직전 집계(시의성) · 정확성 생명 → 오픈일·가격·위치 출처 확인 필수 |

```
/insta-j-room-create-news
이번 주 소식
- OO매장 신규 '테마X' 6/20 오픈, 강남, 공포
- XX매장 오픈할인 20% ~6/30, 홍대
```

## 8. 현재 상태 (수시 업데이트)

- ✅ 인프라: 계정/로고/브랜드/파이프라인 구축 완료
- ✅ **4개 카테고리 렌더러 + 스킬 완성** (추천/소식/후기/연방)
- ✅ 콘텐츠 2개: **PLAY33 추천** + **룸즈에이 메가게임 후기** — 둘 다 업로드 직전 대기
- 📦 재고(ready): **2개 / 목표 4개**
- ⏳ 미정: 발행 시작 시점/빈도 (사용자 검토 중)

> 발행 시작 전 권장: 재고 4개 확보(추천 위주) 후 주 2회. 소식은 발행 직전 집계.
