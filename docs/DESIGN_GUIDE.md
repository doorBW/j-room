# 미리캔버스 카드뉴스 디자인 가이드

> 디자인 초보(서버 개발자)가 따라만 하면 일관된 카드뉴스를 만들 수 있도록.
> 처음 한 번 마스터 템플릿을 만들면, 다음부터는 텍스트만 교체 = 10~20분.

---

## 0. 준비 (한 번만)

1. [미리캔버스](https://www.miricanvas.com) 가입 (무료로 시작)
2. **새 디자인 → 인스타그램 → 세로형 이미지 (1080×1350)** 선택
3. 슬라이드(페이지)를 **8장** 추가 (오른쪽 "+" 또는 페이지 복제)

> 캐러셀은 모든 장이 같은 크기여야 함. 1080×1350 고정.

---

## 1. 브랜드 설정 (한 번만 — 모든 콘텐츠 공통)

### 컬러 팔레트 (공포/미스터리 톤)
미리캔버스에서 색상 입력란에 HEX 그대로 입력:

| 역할 | HEX | 용도 |
|------|-----|------|
| 배경 메인 | `1A1A2E` | 카드 배경 (딥 네이비) |
| 배경 더 어둡게 | `0F0F1E` | 공포 강조 카드 |
| 강조 | `E94560` | 별점, 키워드, 포인트 (크림슨) |
| 텍스트 밝게 | `FFFFFF` | 어두운 배경 위 본문 |
| 텍스트 보조 | `B8B8C8` | 부가 정보 (회색빛) |

> 공포 콘텐츠라 전체적으로 어둡게. 강조는 빨강 하나로 통일하면 세련됨.

### 폰트
- **제목**: Pretendard Bold (또는 "검은고딕", "G마켓산스 Bold")
- **본문**: Pretendard Regular (또는 "프리텐다드", "노토산스")
- 미리캔버스 폰트 검색창에 "프리텐다드" 입력

### 글자 크기 (1080×1350 기준)
| 요소 | 크기 | 비고 |
|------|------|------|
| 표지 대제목 | 70~90pt | 한 줄에 강하게 |
| 카드 제목 | 50~64pt | |
| 소제목 | 36~44pt | |
| 본문 | 30~38pt | **최소 30pt** (모바일 가독성) |
| 캡션/출처 | 24~28pt | 가장 작게 |

### 공통 요소 (모든 카드에)
- **좌상단**: 로고 작게 (60~80px) — `assets/logos/logo-square-v1.png`
- **우하단**: 카드 번호 `3 / 8` (작게, 회색)
- **하단 중앙 또는 모서리**: `@jroom.official` (작게)

> 이 3개를 첫 카드에 배치 → 복사해서 모든 카드에 붙이면 위치 통일됨.

## 2. 마스터 템플릿 만들기 (재사용의 핵심)

레이아웃 **3종**만 만들면 모든 카드 커버됨:

### 레이아웃 A — 표지형 (1장, 8장째 CTA)
```
┌─────────────┐
│ [로고]        │
│               │
│   대제목       │ ← 70~90pt, 중앙
│   (후킹 문구)  │ ← 36pt, 강조색
│               │
│        [배경이미지 어둡게]
│          @jroom · 1/8 │
└─────────────┘
```

### 레이아웃 B — 본문형 (테마 소개 카드들)
```
┌─────────────┐
│ [로고]  🌑 LV.3│ ← 공포도 라벨
│  〈테마명〉     │ ← 56pt
│  "인용 문구"   │ ← 32pt, 기울임
│ ─────────── │
│  장르·난이도·인원│ ← 30pt
│  추천도 ★★★★★ │ ← 강조색
│  한줄평…       │ ← 32pt
│          @jroom · 5/8 │
└─────────────┘
```

### 레이아웃 C — 표/정보형 (비교표, 매장정보)
```
┌─────────────┐
│  제목         │
│ ┌─────────┐ │
│ │  표 / 리스트 │ │ ← 정렬 깔끔하게
│ └─────────┘ │
│          @jroom · 6/8 │
└─────────────┘
```

> 만든 뒤 "내 템플릿"으로 저장 → 다음 콘텐츠는 복제해서 텍스트만 교체.

## 3. 배경 이미지 (AI 생성, 선택)

단색 배경만으로도 충분하지만, 분위기를 살리려면 AI 배경 이미지를 깔고 **어둡게 오버레이**(투명도 40~60% 검정).

### Gemini/Midjourney 프롬프트 (공포 테마용)

**표지/공통 배경**:
```
dark horror escape room interior, ominous mysterious atmosphere,
deep navy blue and crimson red tones, cinematic dramatic lighting,
empty space for text in center, no text, no people, vertical 4:5
```

**그날 (드라마 공포)**:
```
dark eerie abandoned bedroom, dim cold light through curtains,
melancholic unsettling atmosphere, horror drama mood,
muted navy tones, no text, no people, vertical 4:5
```

**목격자 (연출/조명)**:
```
dramatic film-noir lighting, dark mysterious room with strong
shadows and a single light source, cinematic thriller atmosphere,
deep contrast, no text, no people, vertical 4:5
```

**다이얼 (콜센터 공포)**:
```
creepy abandoned call center at night, rows of empty desks,
red emergency light glow, vintage telephone, horror thriller,
ominous deep shadows, no text, no people, vertical 4:5
```

> 생성 이미지는 `assets/images/` 에 저장. 배경 위 텍스트가 안 읽히면 → 검정 반투명 도형을 텍스트 뒤에 깔기.

## 4. 요소 표현법

### 별점 (추천도)
- 텍스트로: `★★★★☆` (채운 별 + 빈 별) — 가장 쉬움
- 미리캔버스 "요소 → 별" 도형으로도 가능
- 강조색(`E94560`)으로 채우면 눈에 띔
- 소수점은 `★★★★☆ 4.5` 처럼 숫자 병기

### 공포도 단계 (이 콘텐츠 시그니처)
- 🌗 입문 / 🌘 중 / 🌑 끝판왕 (달 이모지로 직관적)
- 또는 불꽃 🔥 개수

### 강조 텍스트
- 핵심 단어만 강조색 또는 형광펜(하이라이트) 효과
- 한 카드에 강조는 1~2곳만 (많으면 안 띔)

## 5. 내보내기

1. 우상단 **다운로드 → PNG** 선택
2. **모든 페이지** 선택 (8장 한 번에)
3. 고화질 옵션 체크
4. 파일명: `play33-건대-01.png` ~ `08.png` 순서대로
5. `assets/images/published/` 또는 별도 폴더에 저장

> 인스타 업로드 시 1번부터 순서대로 선택 = 카드 순서 유지.

---

## 6. 발행 전 체크리스트

- [ ] 모든 카드 글자 ≥ 30pt (모바일에서 읽히는지)
- [ ] 1번 카드(표지)만 봐도 클릭하고 싶은지
- [ ] 색상/폰트가 8장 통일됐는지
- [ ] 로고·카드번호·핸들 위치가 일정한지
- [ ] 오탈자 (소리내어 읽기)
- [ ] 매장명/테마명/가격 정확한지
- [ ] 스포일러 없는지
- [ ] 마지막 카드에 CTA 있는지
