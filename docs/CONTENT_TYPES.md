# 콘텐츠 타입 정의

3가지 핵심 콘텐츠 타입의 구조, 카드 수, 작성 규칙을 정의합니다.

---

## 1. 연방 일정 추천 (course)

> "오늘 강남에서 4시간 방탈출 할 사람?" — 동선 + 시간표 + 테마 소개를 한 번에

### 구조 (총 8~10장)

| # | 카드 | 내용 |
|---|------|------|
| 1 | 표지 | "강남역 4시간 X3 방탈출 코스" + 지도 썸네일 |
| 2 | 코스 개요 | 타임라인 (10:00 모임 → 22:00 마무리) |
| 3 | 1번 매장 | 매장명, 테마, 난이도, 소요시간 |
| 4 | 1번 → 2번 동선 | 도보 X분 / 식사 추천 |
| 5 | 2번 매장 | (위와 동일) |
| 6 | 2번 → 3번 동선 | |
| 7 | 3번 매장 | (위와 동일) |
| 8 | 총평 | 추천 인원/난이도 구성/예산 합계 |
| 9 | CTA | "이 코스 저장 + 다음 지역 댓글로!" |

### 작성 시 필수 정보 (사용자가 제공)
- 지역명 (구체적: "강남역 11번 출구 반경 500m")
- 시간 순서 + 각 테마 시작/종료 시각
- 각 테마 난이도 (1~5)
- 식사/카페 추천 (선택)
- 인원 권장 (2/3/4/4+)
- 총 예산 추정

### 차별화 포인트
- **동선의 효율성**: 도보 5분 이내가 베스트
- **난이도 그라데이션**: 가벼움 → 무거움 순서 추천
- **장르 다양성**: 추리/공포/모험 섞임

## 2. 방탈출 후기 (review)

> 직접 체험한 1개 테마에 대한 솔직 추천

### 구조 (총 6~8장)

| # | 카드 | 내용 |
|---|------|------|
| 1 | 표지 | "[매장명] [테마명]" + 한 줄 후킹 |
| 2 | 한눈에 보기 | 난이도 / 공포도 / 추리비중 / 추천도 / 권장인원 (별점 시각화) |
| 3 | 분위기 | 스포 없이 분위기/세계관 설명 |
| 4 | 좋았던 점 | 2~3개 (장치, 공간, 연출 등) |
| 5 | 아쉬웠던 점 | 솔직하게 1~2개 |
| 6 | 추천 대상 | "이런 사람에게" / "이런 사람은 피하세요" |
| 7 | 정보 | 위치, 가격, 소요시간, 예약방법 |
| 8 | CTA | "비슷한 테마 추천받고 싶다면 댓글로!" |

### 별점 기준 (일관성 유지)

| 항목 | 1점 | 5점 |
|------|-----|-----|
| **난이도** | 입문자 OK | 베테랑도 힘듦 |
| **공포도** | 전혀 안 무서움 | 비명 보장 |
| **추리비중** | 활동/액션 위주 | 머리 계속 굴려야 함 |
| **추천도** | 굳이 갈 필요는 | 무조건 가야 함 |

> 별점 기준은 한 번 정하면 일관되게 사용. 인플레이션 주의.

### 스포일러 방지 규칙
- 자물쇠 종류, 힌트, 정답 절대 노출 X
- "이 장면에서 깜짝 놀랐다" 정도는 OK
- 스토리 1/3 시점까지만 언급

## 3. 테마별 추천 (recommendation)

> 큐레이션. "공포 초심자에게 추천하는 5선" 같은 구조.

### 구조 (총 7~9장)

| # | 카드 | 내용 |
|---|------|------|
| 1 | 표지 | "[주제] TOP 5" (예: "4인이 풀기 좋은 추리 방탈출 5선") |
| 2 | 큐레이션 기준 | 어떤 기준으로 골랐는지 (난이도, 인원, 장르) |
| 3 | 1위 | 매장명/테마명 + 한 줄 추천 이유 + 별점 |
| 4 | 2위 | |
| 5 | 3위 | |
| 6 | 4위 | |
| 7 | 5위 | |
| 8 | 비교표 | 5개 한 페이지 비교 (난이도/공포/가격) |
| 9 | CTA | "어떤 게 끌리세요? 댓글로 알려주세요" |

### 큐레이션 주제 풀

**입문자 시리즈**
- "방탈출 처음인데 뭐부터?" 입문 5선
- "공포 빼고 즐길 수 있는" 안전 추천 5선

**상황별 시리즈**
- "데이트하기 좋은" 로맨틱 방탈출
- "회식/팀 빌딩에 좋은" 4~6인 방탈출
- "비 오는 날 가기 좋은" 실내 방탈출

**장르 시리즈**
- 공포 매니아용 진짜 무서운 5선
- 추리/논리 중심 두뇌 풀가동 5선
- 액션/활동성 높은 5선

**지역 시리즈**
- 강남/홍대/건대 각 지역 BEST 5
- 수도권 외 (부산/대구/광주) 명작 5선

**난이도/시간 시리즈**
- 90분 안에 끝나는 가벼운 5선
- 2시간+ 대작 5선

### 큐레이션 원칙
- **본인이 실제로 경험한 것만** (간접 추천은 신뢰도 ↓)
- 같은 매장에서 2개 이상 추천하지 않기 (다양성)
- 5위 안에 최소 1개는 의외의 선택 (덕후들의 관심 유도)

## 콘텐츠 타입 추가 시 가이드

새 타입이 떠오르면 다음을 정의:
1. **목적**: 정보 전달 / 후기 / 가이드 / 토론 유도
2. **카드 수**: 6~10장 (인스타 권장)
3. **카드별 슬롯**: 표지/본문/마무리의 역할 분담
4. **반복 요소**: 별점, 비교표, 지도 등 시각 요소
5. **사용자 부담**: 1편당 작성 시간 추정

추가 후 `templates/card-news/<신규타입>.md` 생성.
