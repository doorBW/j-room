# 브랜드 가이드 (초안)

> 일관된 톤앤매너로 시리즈물 같은 느낌을 주기 위한 가이드.
> 운영하면서 점점 다듬으세요.

---

## 톤앤매너

**페르소나**: 방탈출을 1년에 100개 이상 풀어본 덕후 친구
**말투**: 반말 X, 친근한 존댓말 ("~해요", "~했어요")
**감정**: 들떠 있지만 과장 X, 솔직함을 강점으로

### 사용 단어 가이드

| 추천 | 비추천 | 이유 |
|------|--------|------|
| "추천드려요" | "강추!!" | 과장 자제 |
| "아쉬웠던 점" | "단점" | 부드러운 표현 |
| "기분 좋게 풀었어요" | "킹갓 명작" | 인터넷 밈 자제 |
| "스포 없이" | "스포일러 주의" | 짧고 자연스럽게 |

## 컬러 팔레트 (예시 — 본인이 정해주세요)

방탈출의 미스터리/긴장감을 표현하면서도 가독성을 잃지 않는 조합:

| 역할 | HEX | 설명 |
|------|-----|------|
| Primary | `#1A1A2E` | 깊은 네이비 (배경/타이틀) |
| Accent | `#E94560` | 진한 핑크-레드 (강조/별점) |
| Soft | `#F5F5F5` | 오프 화이트 (본문 배경) |
| Text Dark | `#0F0F1E` | 본문 텍스트 |
| Text Light | `#FFFFFF` | 어두운 배경 위 텍스트 |

> AI 이미지 생성 시 컬러 키워드 함께 넣기: "deep navy, crimson red accent, mysterious atmosphere"

## 시각 요소 일관성

### 표지 공통 요소
- 좌상단 또는 우상단에 작은 로고 (있다면)
- 하단에 콘텐츠 타입 라벨 (예: "REVIEW #042")
- 일관된 위치의 그라데이션 또는 오버레이

### 별점 표현
- ⭐ 이모지 또는 SVG 5개 (꽉찬 별 + 빈 별)
- 항상 같은 위치 (오른쪽 정렬 등)

### 매장명/테마명 표기
- 매장명은 **bold**, 테마명은 일반 (또는 따옴표)
- 예: **제로월드 강남** 〈비밀의 화원〉

---

## 콘텐츠 안전/윤리

- **악평 자제**: 부정적 후기는 "이런 분에게는 안 맞을 수 있어요" 식으로 완곡하게
- **유료 협찬은 명시**: #광고 #협찬 해시태그 + 캡션 첫줄
- **사진 권리**: 매장에서 사진 촬영 가능한지 확인 (대부분 외부만 OK)
- **스포일러 금지**: 절대선
