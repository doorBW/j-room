# 인스타그램 캡션 템플릿

콘텐츠 타입별 캡션 + 해시태그 모음.

## 캡션 구조 원칙

1. **첫 1줄**: 모바일 미리보기에 보이는 영역. 후킹 강하게.
2. **2~4줄**: 공백 줄로 분리, 핵심 메시지
3. **이모지**: 1~2개만 (과하면 가독성 ↓)
4. **CTA**: 저장/댓글/공유 중 하나 명확히
5. **해시태그**: 캡션 본문과 분리 (`—` 또는 공백 3줄)
